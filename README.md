# 📦 Gerenciador de Estoque em Python

## 📖 Sobre

Projeto desenvolvido em Python para gerenciamento de estoque utilizando dicionários, funções e armazenamento em arquivos JSON.

Os dados são salvos automaticamente, permitindo que o estoque seja recuperado sempre que o programa for iniciado.

## 🚀 Funcionalidades

- ✅ Adicionar produtos
- ✅ Listar produtos
- ✅ Buscar produto
- ✅ Atualizar quantidade
- ✅ Remover produtos
- ✅ Salvar dados em JSON
- ✅ Carregar dados automaticamente

## 🛠️ Tecnologias

- Python 3
- JSON

```bash
python gerenciador_de_estoque.py
